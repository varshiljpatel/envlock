---
title: Installation
description: A complete guide to install Envlock.
---

You can install Envlock using your favorite package manager:

import { Tabs, TabItem } from "@astrojs/starlight/components";

<Tabs syncKey="package_manager">
  <TabItem label="npm">
    ```
    npm install -g envlock
    ```
  </TabItem>
  <TabItem label="yarn">
    ```
    yarn global add envlock
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```
    pnpm add -g envlock 
    ```
  </TabItem>
  <TabItem label="bun">
    ```
    bun add -g envlock 
    ```
  </TabItem>
</Tabs>

> :::note 
> This will add envlock as your global dependency.
